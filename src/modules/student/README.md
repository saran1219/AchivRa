# Student Achievement Management Module

This module allows students to submit and manage their achievement details.

## Features
- Create new achievement records
- Upload certificates (PDF/Image)
- Update achievement details
- View achievement status
- Track submission history

## Files Structure
- `studentService.ts` - Student achievement service
- `useAchievements.ts` - Achievement management hook
- `AchievementForm.tsx` - Form component for adding achievements
- `AchievementList.tsx` - List component for viewing achievements
- `types.ts` - Type definitions
