# Admin Management Module

This module provides administrators with tools to manage users, categories, and system monitoring.

## Features
- Manage users and roles
- Manage achievement categories
- View department-wise achievements
- Generate system reports
- Monitor system health

## Files Structure
- `adminService.ts` - Admin service
- `useAdmin.ts` - Admin management hook
- `UserManagement.tsx` - User management component
- `CategoryManagement.tsx` - Category management component
- `ReportGenerator.tsx` - Report generation component
- `types.ts` - Type definitions
