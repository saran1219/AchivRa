# Dashboard Module

This module provides summarized system data with analytics and statistics.

## Features
- View achievement statistics
- Filter achievements by department or category
- Export data summaries
- Real-time status indicators
- Custom report generation

## Files Structure
- `dashboardService.ts` - Dashboard data service
- `useDashboard.ts` - Dashboard hook
- `StatisticsCard.tsx` - Statistics component
- `AnalyticsChart.tsx` - Chart component
- `types.ts` - Type definitions
