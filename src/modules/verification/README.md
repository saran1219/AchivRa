# Verification & Approval Module

This module enables faculty members to review, verify, and approve student achievements.

## Features
- View submitted achievements
- Approve or reject achievements
- Add verification remarks
- Update achievement status
- Generate verification reports

## Files Structure
- `verificationService.ts` - Verification service
- `useVerification.ts` - Verification hook
- `VerificationQueue.tsx` - Queue component
- `VerificationDetail.tsx` - Detail view component
- `types.ts` - Type definitions
